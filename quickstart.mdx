The Pulumi AWS Provider lets you define and deploy AWS infrastructure using real programming languages. This quickstart walks you through creating a Pulumi project, configuring credentials, writing a resource, and running your first deployment — from zero to a live S3 bucket in minutes.

## Prerequisites

Before you begin, make sure you have:

- [Pulumi CLI installed](https://www.pulumi.com/docs/get-started/install/) (`pulumi version` should return v3 or higher)
- An AWS account with an IAM user that has sufficient permissions
- AWS credentials configured (see [Installation & Configuration](/installation-configuration))

<Note>
  If you haven't configured AWS credentials yet, run `aws configure` with your access key and secret key, or set `AWS_ACCESS_KEY_ID` and `AWS_SECRET_ACCESS_KEY` environment variables.
</Note>

## Deploy your first resource

<Steps>
  <Step title="Create a new Pulumi project">
    Use `pulumi new` to scaffold a new project with the AWS template for your language:

    <CodeGroup>
    ```bash TypeScript
    pulumi new aws-typescript
    ```
    ```bash Python
    pulumi new aws-python
    ```
    ```bash Go
    pulumi new aws-go
    ```
    ```bash .NET
    pulumi new aws-csharp
    ```
    </CodeGroup>

    The wizard will ask for a project name, description, and your AWS region. Accept the defaults or customize as needed.

    <Tip>
      You can also start from scratch with `pulumi new` and select the AWS template interactively.
    </Tip>
  </Step>

  <Step title="Install the provider package">
    If you're adding the provider to an existing project, install it with your language's package manager:

    <CodeGroup>
    ```bash TypeScript/JavaScript
    npm install @pulumi/aws
    ```
    ```bash Python
    pip install pulumi_aws
    ```
    ```bash Go
    go get github.com/pulumi/pulumi-aws/sdk/v7
    ```
    ```bash .NET
    dotnet add package Pulumi.Aws
    ```
    </CodeGroup>
  </Step>

  <Step title="Write your first resource">
    Replace the contents of your main program file with a simple S3 bucket with versioning enabled:

    <CodeGroup>
    ```typescript TypeScript (index.ts)
    import * as pulumi from "@pulumi/pulumi";
    import * as aws from "@pulumi/aws";

    // Create an S3 bucket
    const bucket = new aws.s3.Bucket("my-bucket", {
        tags: {
            Environment: "dev",
        },
    });

    // Enable versioning on the bucket
    const versioning = new aws.s3.BucketVersioning("my-bucket-versioning", {
        bucket: bucket.id,
        versioningConfiguration: {
            status: "Enabled",
        },
    });

    // Export the bucket name and ARN
    export const bucketName = bucket.id;
    export const bucketArn = bucket.arn;
    ```
    ```python Python (__main__.py)
    import pulumi
    import pulumi_aws as aws

    # Create an S3 bucket
    bucket = aws.s3.Bucket("my-bucket",
        tags={
            "Environment": "dev",
        }
    )

    # Enable versioning on the bucket
    versioning = aws.s3.BucketVersioning("my-bucket-versioning",
        bucket=bucket.id,
        versioning_configuration=aws.s3.BucketVersioningVersioningConfigurationArgs(
            status="Enabled",
        )
    )

    # Export the bucket name and ARN
    pulumi.export("bucket_name", bucket.id)
    pulumi.export("bucket_arn", bucket.arn)
    ```
    ```go Go (main.go)
    package main

    import (
        "github.com/pulumi/pulumi-aws/sdk/v7/go/aws/s3"
        "github.com/pulumi/pulumi/sdk/v3/go/pulumi"
    )

    func main() {
        pulumi.Run(func(ctx *pulumi.Context) error {
            // Create an S3 bucket
            bucket, err := s3.NewBucket(ctx, "my-bucket", &s3.BucketArgs{
                Tags: pulumi.StringMap{
                    "Environment": pulumi.String("dev"),
                },
            })
            if err != nil {
                return err
            }

            // Enable versioning
            _, err = s3.NewBucketVersioning(ctx, "my-bucket-versioning", &s3.BucketVersioningArgs{
                Bucket: bucket.ID(),
                VersioningConfiguration: &s3.BucketVersioningVersioningConfigurationArgs{
                    Status: pulumi.String("Enabled"),
                },
            })
            if err != nil {
                return err
            }

            ctx.Export("bucketName", bucket.ID())
            ctx.Export("bucketArn", bucket.Arn)
            return nil
        })
    }
    ```
    ```csharp .NET (Program.cs)
    using Pulumi;
    using Aws = Pulumi.Aws;

    return await Deployment.RunAsync(() =>
    {
        // Create an S3 bucket
        var bucket = new Aws.S3.Bucket("my-bucket", new Aws.S3.BucketArgs
        {
            Tags = new InputMap<string>
            {
                { "Environment", "dev" },
            },
        });

        // Enable versioning
        var versioning = new Aws.S3.BucketVersioning("my-bucket-versioning", new Aws.S3.BucketVersioningArgs
        {
            Bucket = bucket.Id,
            VersioningConfiguration = new Aws.S3.Inputs.BucketVersioningVersioningConfigurationArgs
            {
                Status = "Enabled",
            },
        });

        return new Dictionary<string, object?>
        {
            ["bucketName"] = bucket.Id,
            ["bucketArn"] = bucket.Arn,
        };
    });
    ```
    </CodeGroup>
  </Step>

  <Step title="Preview and deploy">
    Run `pulumi up` to preview the changes and deploy:

    ```bash
    pulumi up
    ```

    Pulumi shows a preview of the resources it will create. Confirm by selecting `yes`:

    ```
    Previewing update (dev)

    Type                            Name                       Plan
    +   pulumi:pulumi:Stack         my-project-dev             create
    +   ├─ aws:s3:Bucket            my-bucket                  create
    +   └─ aws:s3:BucketVersioning  my-bucket-versioning       create

    Resources:
        + 2 to create

    Do you want to perform this update? yes
    Updating (dev)

    Type                            Name                       Status
    +   pulumi:pulumi:Stack         my-project-dev             created
    +   ├─ aws:s3:Bucket            my-bucket                  created
    +   └─ aws:s3:BucketVersioning  my-bucket-versioning       created

    Outputs:
        bucketArn  = "arn:aws:s3:::my-bucket-a1b2c3d4"
        bucketName = "my-bucket-a1b2c3d4"

    Resources:
        + 2 created

    Duration: 8s
    ```

    <Tip>
      Run `pulumi preview` first if you want to see the plan without deploying.
    </Tip>
  </Step>

  <Step title="Clean up">
    When you're done, destroy the resources to avoid incurring AWS charges:

    ```bash
    pulumi destroy
    ```

    <Warning>
      `pulumi destroy` permanently deletes all resources in the stack. Make sure you don't need them before running this command.
    </Warning>
  </Step>
</Steps>
