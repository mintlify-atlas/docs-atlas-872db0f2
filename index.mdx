## Get started in three steps

<Steps>
  <Step title="Install the provider">
    Add the Pulumi AWS provider to your project using your language's package manager.

    <CodeGroup>
    ```bash TypeScript/JavaScript
    npm install @pulumi/aws
    ```
    ```bash Python
    pip install pulumi-aws
    ```
    ```bash Go
    go get github.com/pulumi/pulumi-aws/sdk/v7
    ```
    ```bash .NET
    dotnet add package Pulumi.Aws
    ```
    </CodeGroup>
  </Step>
  <Step title="Configure AWS credentials">
    Set your AWS region and credentials so Pulumi can authenticate to your account.

    ```bash
    export AWS_ACCESS_KEY_ID=<YOUR_ACCESS_KEY_ID>
    export AWS_SECRET_ACCESS_KEY=<YOUR_SECRET_ACCESS_KEY>
    export AWS_REGION=us-east-1
    ```

    Or use the Pulumi CLI to set your region:

    ```bash
    pulumi config set aws:region us-east-1
    ```
  </Step>
  <Step title="Deploy your first resource">
    Create an S3 bucket with a few lines of code and run `pulumi up` to deploy.

    <CodeGroup>
    ```typescript TypeScript
    import * as aws from "@pulumi/aws";

    const bucket = new aws.s3.Bucket("my-bucket");

    export const bucketName = bucket.id;
    ```
    ```python Python
    import pulumi
    import pulumi_aws as aws

    bucket = aws.s3.Bucket("my-bucket")

    pulumi.export("bucket_name", bucket.id)
    ```
    ```go Go
    package main

    import (
        "github.com/pulumi/pulumi-aws/sdk/v7/go/aws/s3"
        "github.com/pulumi/pulumi/sdk/v3/go/pulumi"
    )

    func main() {
        pulumi.Run(func(ctx *pulumi.Context) error {
            bucket, err := s3.NewBucket(ctx, "my-bucket", &s3.BucketArgs{})
            if err != nil {
                return err
            }
            ctx.Export("bucketName", bucket.ID())
            return nil
        })
    }
    ```
    </CodeGroup>

    ```bash
    pulumi up
    ```
  </Step>
</Steps>
